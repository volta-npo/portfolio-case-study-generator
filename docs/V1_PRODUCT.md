# Portfolio Case Study Generator v1.0 Product Specification

## Workbench

Ethical Portfolio Studio

## Domain table

Case study assets

## Metrics

- Publish Readiness
- Consent Safety
- Impact Proof

## Domain rows

- Student role captured
- Problem/intervention written
- Before proof attached
- After proof attached
- Outcome metric recorded
- Client consent/anonymization set
- Reflection completed
- Case study exported

## Generated deliverables

- Markdown case study
- PDF-ready case study
- LinkedIn summary

## Validation gates

- Consent or anonymization required
- Private data redacted
- At least one measurable outcome
